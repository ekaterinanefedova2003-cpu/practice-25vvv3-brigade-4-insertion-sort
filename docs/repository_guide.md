# Как пользоваться репозиторием

README.md - общее описание проекта.
src/main.py - исходный код программы.
tests/input_example.txt - пример входных данных.
docs/testing.md - описание проверки программы.
docs/bachelor_report.docx - отчет бакалавров, который загружается отдельным Pull Request.

Основная ветка: main.
Все изменения передаются на проверку через Pull Request.
