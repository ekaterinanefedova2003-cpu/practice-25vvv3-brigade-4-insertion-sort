# Практика 25ВВВ3, бригада 4
Репозиторий создан для организации работы бригады 4 группы 25ВВВ3 по учебной практике.

## Тема работы
Разработка программы сортировки вставками Insertion Sort с графическим интерфейсом.

## Участники и роли
Шебалкин  
Роль: разработка алгоритма сортировки и части программной реализации.  
Основные задачи: алгоритм сортировки, проверка корректности сортировки, подготовка кода.

Юткина  
Роль: тестирование, оформление результатов и загрузка отчета бакалавров.  
Основные задачи: проверка программы, оформление результатов тестирования, загрузка отчета.

## Структура репозитория
`src/main.py` - исходный код программы.  
`tests/input_example.txt` - пример входных данных.  
`docs/testing.md` - описание тестирования.  
`docs/repository_guide.md` - инструкция по ориентированию в репозитории.  
`docs/bachelor_report.docx` - отчет бакалавров, загружается отдельным Pull Request.

## Организация работы
Основная ветка проекта: `main`.
Изменения не добавляются напрямую в `main`. Для каждой задачи используется отдельная ветка и Pull Request.

## Основные задачи
1. Разработка алгоритма сортировки.
2. Разработка интерфейса.
3. Тестирование программы.
4. Загрузка отчета бакалавра.
5. Загрузка файла с инструкцией по ориентированию в репозитории.

## Как запустить программу
1. Установить Python 3.
2. Открыть терминал в папке проекта.
3. Перейти в папку `src`.
4. Выполнить команду:

```bash
python main.py
```

Если команда не работает:

```bash
py main.py
```
